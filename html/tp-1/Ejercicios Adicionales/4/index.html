Uso del Elemento <iframe>

Ejemplo Básico:

El siguiente código muestra cómo se utiliza el elemento <iframe> para incrustar otra página web o documento dentro de la página actual.

            <!DOCTYPE html>
            <html lang="es">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Ejemplo de iframe</title>
            </head>
            <body>

                <h1>Página Principal</h1>

                <p>Contenido de la página principal.</p>

                <!-- Ejemplo de iframe -->
                <iframe src="pagina_incrustada.html" width="600" height="400" title="Contenido Incrustado"></iframe>

            </body>
            </html>
